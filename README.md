# 🎮 Tic-Tac-Toe

A simple and responsive Tic-Tac-Toe game built using **HTML**, **CSS**, and **JavaScript**.

## 🚀 Features

- ✅ Two-player gameplay (X vs O)
- ✅ Winner detection
- ✅ Draw detection
- ✅ Reset Game button
- ✅ New Game button
- ✅ Responsive design
- ✅ Clean and modern UI

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript

## 🎯 How to Run

1. Clone the repository

```bash
git clone https://github.com/ahwan84/Tic-Tac-Toe.git
```

2. Open the project folder.

3. Double-click `index.html`

OR

Open it using **Live Server** in VS Code.
